Tools needed for compiling scss to css

  • Node: js environment

    • nodejs.org - click install
    • open terminal
    • node -v
    • npm -v

  • npm: node package manager

    • npm init
    • package.json
    • npm --save-dev {package-name}
    • npm --save {package-name}
    • npm install -g {package-name}
    • npm install
    • node_modules folder

  • Gulp.js - Automated task runner for node

    • npm install -g gulp (needs admin previlages)
    • npm install
    • gulp tasks
      • gulp scss
      • gulp watch
      • gulp
